# 🍽️ Eat Curious  (Work in Progress) 🚧

**Eat Curious** is a modern, animated food website built with **GSAP** for smooth animations and dynamic interactions. It offers a visually engaging experience for food lovers, featuring animated transitions, sleek UI, and a responsive design. 🚀✨  

## 🚀 Features  
- 🍔 Stunning food-related animations with **GSAP**  
- 🎨 Modern & interactive UI/UX  
- 📱 Fully responsive design  
- ⚡ Smooth page transitions & effects  
- 🖼️ Swiper.js for interactive sliders  

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** GSAP, Swiper.js, Bootstrap

## 📌 Installation & Setup  

### 🔽 Download as ZIP  
1. Go to the [GitHub repository](https://github.com/sujitbarui/Eat-Curious).  
2. Click on the green **Code** button.  
3. Select **Download ZIP**.  
4. Extract the ZIP file on your computer.  
5. Open `index.html` in your browser or use Live Server for development.  

### 🛠 Clone the Repository  
1. Clone the repository:  
   ```sh
   git clone https://github.com/sujitbarui/Eat-Curious.git

2. Navigate to the project folder:
   ```sh
   cd Eat-Curious
   ```
3. Open `index.html` in your browser or use Live Server for development.
